# 19. Two Sum
 
## 目標 Goals
- 給定一個整數陣列 nums 和一個整數 target
- 印出 nums 陣列的數字中，兩數和為 target 的 index
- 假設每組輸入只有一個解決方法，且同一個元素不能使用 2 次
- 可使用任意順序回傳答案 

## 要求 Requirements
- 使用者輸入 #01：nums = [2,7,11,15], target = 9 
- 印出 #01：[0,1]
- 使用者輸入 #02：nums = [3,2,4], target = 6
- 印出 #02：[1,2]
