# 08. Merge Sets

## 目標 Goals
- 將不同的 dict，合併成一個 dict

## 要求 Requirements
- 使用者輸入：d1 = {1:10, 2:20}, d2 = {3:30, 4:40}, d3 = {5:50, 6:60}
- 印出：{1: 10, 2: 20, 3: 30, 4: 40, 5: 50, 6: 60}
